# AWS-Cloud-Foundations
# Desafio DIO - AWS Cloud Foundations

📖 Sobre

Este repositório foi criado para documentar os conhecimentos adquiridos durante o laboratório da formação AWS Cloud Foundations da DIO.

📚 Conteúdos estudados

Durante as aulas foram abordados os seguintes temas:

- Conceitos de computação em nuvem (Cloud Computing);
- Modelos de serviço:
  - IaaS (Infrastructure as a Service);
  - PaaS (Platform as a Service);
  - SaaS (Software as a Service);
- Introdução ao Amazon EC2;
- Amazon EBS (Elastic Block Store) e seu uso como armazenamento para instâncias;
- Organização de recursos na AWS;
- Usuários e grupos no IAM para gerenciamento de permissões;
- Boas práticas de segurança e controle de acesso;
- Avisos de gastos (Billing Alerts) para monitorar custos e evitar cobranças inesperadas.

💡 Aprendizados

Com este laboratório foi possível compreender:

- A diferença entre IaaS, PaaS e SaaS;
- A função do Amazon EC2 dentro da infraestrutura da AWS;
- A importância do Amazon EBS para armazenamento persistente;
- Como usuários e grupos facilitam o gerenciamento de permissões na AWS;
- A importância de configurar alertas de custos para acompanhar os gastos na nuvem.
